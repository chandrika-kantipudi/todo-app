### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/chandrika-kantipudi/todo-app.git
   cd todo-app
```

2. **Set up the backend**
```bash
   cd backend
   npm install
   npm start
```

3. **Set up the frontend**
```bash
   cd ../frontend
   npm install
   npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
